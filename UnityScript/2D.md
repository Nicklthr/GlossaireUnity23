## 2D
OnTriggerEnter2D : Fonction qui se declenche lorsqu'un Collider2D externe entre dans la zone de detection du Collider2D de l'objet. Elle reçoit en parametre le Collider2D qui est entré en collision.
```public void OnTriggerEnter2D( Collider2D collision ) { }```
OnTriggerStay2D : Fonction qui se declenche à chaque frame quand un Collider2D externe est dans la zone de detection du Collider2D de l'objet. Elle reçoit en parametre le Collider2D qui est entré en collision.
```public void OnTriggerStay2D( Collider2D collision ) { }```
OnTriggerExit2D : Fonction qui se declenche lorsqu'un Collider2D externe sort dans la zone de detection du Collider2D de l'objet. Elle reçoit en parametre le Collider2D qui est entré en collision.
```public void OnTriggerExit2D( Collider2D collision ) { }```
OnCollisionEnter2D : Fonction qui se declanche lorsqu'un Collider2D externe entre en collision avec le Collider2D de l'objet. Elle reçoit en parametre le Collider2D qui est entré en collision.
```public void OnCollisionEnter2D( Collider2D collision ) { }```
OnCollisionStay2D : Fonction qui se declanche à chaque frame quand un Collider2D externe est en collision avec le Collider2D de l'objet. Elle reçoit en parametre le Collider2D qui est entré en collision.
```public void OnCollisionStay2D( Collider2D collision ) { }```
OnCollisionExit2D : Fonction qui se declanche lorsqu'un Collider2D externe sort d'une collision avec le Collider2D de l'objet. Elle reçoit en parametre le Collider2D qui est entré en collision.
```public void OnCollisionExit2D( Collider2D collision ) { }```
