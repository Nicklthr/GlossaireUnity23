## 3D
OnTriggerEnter : Fonction qui se declenche lorsqu'un Collider externe entre dans la zone de detection du Collider de l'objet. Elle reçoit en parametre le Collider qui est entré en collision.
```public void OnTriggerEnter( Collider collision ) { }```
OnTriggerStay : Fonction qui se declenche à chaque frame quand un Collider externe est dans la zone de detection du Collider de l'objet. Elle reçoit en parametre le Collider qui est entré en collision.
```public void OnTriggerStay( Collider collision ) { }```
OnTriggerExit : Fonction qui se declenche lorsqu'un Collider externe sort dans la zone de detection du Collider de l'objet. Elle reçoit en parametre le Collider qui est entré en collision.
```public void OnTriggerExit( Collider collision ) { }```
OnCollisionEnter : Fonction qui se declanche lorsqu'un Collider externe entre en collision avec le Collider de l'objet. Elle reçoit en parametre le Collider qui est entré en collision.
```public void OnCollisionEnter( Collider collision ) { }```
OnCollisionStay : Fonction qui se declanche à chaque frame quand un Collider externe est en collision avec le Collider de l'objet. Elle reçoit en parametre le Collider qui est entré en collision.
```public void OnCollisionStay( Collider collision ) { }```
OnCollisionExit : Fonction qui se declanche lorsqu'un Collider externe sort d'une collision avec le Collider de l'objet. Elle reçoit en parametre le Collider qui est entré en collision.
```public void OnCollisionExit( Collider collision ) { }```
