##While : utilisée lorsque l'on ne connait pas le nombre d'iteration
---

```C#
while (variable < x)
	{
		code a appliquer
		variable ++
	}
```

for : utilisée lorsque l'on connait le nombre d'iteration

```C#
for (i = 0; i <=10; i++)
	{
		code à appliquer
	}
```

for each : utilisée pour parcourir toutes les lignes d'un tableau

```C#
foreach (variable in tableau[])
	{
		code a appliquer
	}
```

do while : a utiliser lorsque lon veux appliquer au moins une fois la condition
```C#
do
	{
		code à appliquer
		variable++
	}
	while (variable <x)
```
