## Opérateurs booléens


">" : Strictement supérieur à


```
var1 > 1
```


">=" : Supérieur ou égal à


```
var1 >= 1
```


== : est similaire à


```
var1 == 1
```


!= : est different de


```
var1 != 1
```


<= : Inférieur ou égal à


```
var1 <= 1
```


< : Strictement inférieur à


```
var1 < 1
```


&& : ET logique (vérifie si deux conditions sont toutes deux vraies)


```
var1 && var2
```


|| : OU logique (vérifie si au moins une des deux conditions est vraie)


```
var1 || var2
```
